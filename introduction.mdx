# Welcome to DIDG System

DIDG System is a modern SaaS platform designed specifically for engineering students and teaching assistants. It provides a comprehensive suite of tools for managing academic resources, organizing teaching assistant sessions (ayudantías), and accessing helpful utilities for students.

## Who Is This For?

DIDG System serves two primary user groups:

**Engineering Students**
- Access course materials and resources organized by subject
- View timeline of past and upcoming teaching assistant sessions
- Bookmark favorite resources and ayudantías for quick access
- Track academic grades and calculate potential outcomes
- Use built-in tools like the grade simulator
- Navigate quickly using the command terminal (Ctrl+K)

**Teaching Assistants**
- Manage course materials and resources
- Schedule and organize ayudantía sessions with video links
- Upload grades and track student progress
- Share resources with students efficiently
- Receive real-time notifications via Telegram
- Secure administrative actions with 2FA

## Technology Stack

DIDG System is built with modern, production-ready technologies:

- **Framework**: Next.js 14 with App Router and TypeScript
- **Styling**: Tailwind CSS, Shadcn/ui components, Framer Motion animations
- **Backend**: Supabase (PostgreSQL, Authentication, Row Level Security)
- **Integrations**: Telegram Bot API for real-time notifications
- **State Management**: React Context API with Server Actions

<Note>
The platform follows clean architecture principles and implements SOLID design patterns for maintainability and scalability.
</Note>

## Architecture Highlights

### Clean Architecture
The codebase is organized following clean architecture principles:

```
src/
├── app/                    # Next.js App Router pages

```

### Server Actions
The platform uses Next.js Server Actions for secure backend operations:

```typescript
// Example: Toggle bookmark on a resource
export async function toggleBookmark(resourceId: string) {
  const supabase = await createClient();
  const { data: { user } } = await supabase.auth.getUser();
  
  if (!user) return { ok: false, message: "No autenticado" };

  const { data: existing } = await supabase
    .from("resource_bookmarks")
    .select("*")
    .eq("user_id", user.id)
    .eq("resource_id", resourceId)
    .single();

  if (existing) {
    await supabase
      .from("resource_bookmarks")
      .delete()
      .eq("user_id", user.id)
      .eq("resource_id", resourceId);
  } else {
    await supabase
      .from("resource_bookmarks")
      .insert({ user_id: user.id, resource_id: resourceId });
  }

  revalidatePath("/resources");
  revalidatePath("/profile");
  return { ok: true, isBookmarked: !existing };
}
```
