### Calculation Logic

**Source:** `GradeSimulator.tsx:26-64`

The simulator uses real-time calculation with two key metrics:

#### 1. Current Average

Calculates your current grade based on completed evaluations:

```typescript
let totalWeightedScore = 0;
let totalWeightUsed = 0;

grades.forEach(g => {
  if (g.grade !== null) {
    totalWeightedScore += g.grade * (g.weight / 100);
    totalWeightUsed += g.weight;
  }
});

if (totalWeightUsed > 0) {
  const currentAverage = totalWeightedScore / (totalWeightUsed / 100);
  // Example: If you have 5.0 in 50% of the course
  // currentAverage = (5.0 * 0.5) / 0.5 = 5.0
}
```

#### 2. Required Grade

Calculates what you need on remaining evaluations to hit your target:

```typescript
const totalDefinedWeight = grades.reduce((sum, g) => sum + g.weight, 0);

if (totalDefinedWeight < 100) {
  const remaining = 100 - totalDefinedWeight;
  // Formula: (Target - CurrentAccumulated) / RemainingWeight%
  const scoreNeeded = (targetGrade - totalWeightedScore) / (remaining / 100);
  
  // Example: Target 4.0, have 2.0 accumulated from 50%
  // Need: (4.0 - 2.0) / 0.5 = 4.0 in remaining 50%
}
```

```typescript
React.useEffect(() => {
  const down = (e: KeyboardEvent) => {
    if (e.key === "k" && (e.metaKey || e.ctrlKey)) {
      e.preventDefault();
      setOpen((open) => !open);
    }
  };
  document.addEventListener("keydown", down);
  return () => document.removeEventListener("keydown", down);
}, []);
```

### Code Execution

**API Integration** (`CodeTerminal.tsx:52-78`)

Uses the [Piston API](https://github.com/engineer-man/piston) for code execution:

```typescript
const runCode = async () => {
  setIsRunning(true);
  
  try {
    const response = await fetch("https://emkc.org/api/v2/piston/execute", {
      method: "POST",
      headers: { "Content-Type": "application/json" },
      body: JSON.stringify({
        language: activeLang.id,
        version: activeLang.version,
        files: [{ content: code }],
      }),
    });

    const data = await response.json();
    
    if (data.run) {
      setOutput(data.run.stdout || data.run.stderr || "Sin salida (Exit Code 0)");
    } else {
      setOutput("Error: No se pudo conectar al compilador.");
    }
  } catch (error) {
    setOutput("Error de red. Intenta nuevamente.");
  } finally {
    setIsRunning(false);
  }
};
```

<ParamField path="language" type="string" required>
  Programming language ID ("python", "javascript", etc.)
</ParamField>

<ParamField path="version" type="string" required>
  Language version to use
</ParamField>

<ParamField path="files" type="array" required>
  Array with single object containing code `content`
</ParamField>

## Best Practices

### Grade Simulator
1. **Start with known weights**: Input evaluation weights first
2. **Enter grades as you go**: Update after each evaluation
3. **Check total weight**: Ensure it sums to 100%
4. **Set realistic targets**: Use 4.0 for passing, 5.0+ for honors

### Command Palette
1. **Learn the shortcut**: `Ctrl/Cmd + K` becomes second nature
2. **Use search**: Type to filter commands instantly
3. **Explore categories**: Discover all available actions
4. **Mobile access**: Use floating button on touch devices

### Code Terminal
1. **Test snippets quickly**: Perfect for algorithm practice
2. **Compare languages**: Switch between languages easily
3. **Save your work**: Copy code before switching languages
4. **Check errors**: Read stderr output for debugging
